# Gestión de Servicios Fotográficos

Este proyecto es una aplicación de consola en Python que permite gestionar servicios fotográficos. Los servicios incluyen información como el nombre del paquete, el precio y una descripción del tipo de evento. Los datos se almacenan en un archivo JSON llamado `datos.json`.

## Características

- **Agregar servicios fotográficos:** Permite añadir un nuevo servicio con nombre, precio y descripción.
- **Editar servicios fotográficos:** Permite modificar los detalles de un servicio existente.
- **Eliminar servicios fotográficos:** Permite eliminar un servicio existente.
